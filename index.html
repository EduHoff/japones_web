O que é o projeto japonês web?

O projeto Japonês Web foi desenvolvido com o propósito de ajudar na familiarização e memorização dos principais 46 símbolos presentes nos dois alfabetos do idioma: o Hiragana e o Katakana.

O objetivo principal do projeto Japonês Web é criar um ambiente acessível onde os estudantes de japonês possam praticar os seus estudos a cerca do sistema de escrita japonesa que, querendo ou não, é intimidador para aqueles que querem começar.
